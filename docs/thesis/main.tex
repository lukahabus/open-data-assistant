% Abstract in Croatian
\begin{sazetak}
U radu je razvijen inteligentni sustav za analizu metapodataka otvorenih skupova podataka koji koristi Retrieval-Augmented Generation (RAG) arhitekturu s velikim jezičnim modelima. Sustav omogućava korisnicima postavljanje pitanja na prirodnom jeziku i automatsko generiranje SPARQL upita.

Ključna inovacija je vektorska baza podataka koja semantički indeksira primjere upita i metapodatke, omogućavajući učenje iz prethodnih iskustava. Sustav implementira multimodalni pristup kombiniranjem RAG-generiranih SPARQL upita, REST API poziva i pretrage sličnih skupova podataka.

Evaluacija pokazuje 92\% točnost u generiranju SPARQL upita, što je poboljšanje u odnosu na tradicionalne pristupe (65\%) i generičke pristupe velikih jezičnih modela (78\%). Prosječno vrijeme odziva od 8.3 sekunde i testiranje na 100 scenarija potvrđuje praktičnu uporabljivost sustava.
\end{sazetak}

% Abstract in English
\begin{abstract}
This thesis presents an intelligent system for open dataset metadata analysis using Retrieval-Augmented Generation (RAG) architecture with large language models. The system enables users to pose natural language questions and automatically generates corresponding SPARQL queries.

The key innovation is a vector database that semantically indexes query examples and metadata, enabling learning from previous experiences. The system implements a multi-modal approach combining RAG-generated SPARQL queries, REST API calls, and similar dataset searches.

Evaluation demonstrates 92\% accuracy in generating correct SPARQL queries, improving over traditional approaches (65\%) and generic large language model approaches (78\%). An average response time of 8.3 seconds and testing on 100 scenarios confirms practical system usability.
\end{abstract>
